## ğŸ“– Overview

**S2P Converter** is a specialized developer tool designed to accelerate the migration from Selenium/Java to Playwright/TypeScript. Instead of strict AST parsing which often fails on partial snippets, S2P uses a **Regex-Heuristic Engine** to identify common patterns, variable declarations, assertions, and WebDriver commands, translating them into idiomatic Playwright syntax.

Built with a **3-Layer Architecture** (Architecture, Navigation, Tools), it ensures reliability and ease of extension.
